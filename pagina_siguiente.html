HTML (HyperText Markup Language) es el lenguaje de marcado estándar utilizado para crear páginas web. Permite estructurar el contenido de una página web y definir su presentación. Es esencial para el desarrollo web.

HTML se compone de etiquetas que indican al navegador cómo mostrar el contenido. Estas etiquetas pueden utilizarse para crear encabezados, párrafos, listas, enlaces y más.
